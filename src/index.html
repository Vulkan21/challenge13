О проекте

Этот проект демонстрирует создание современного статического сайта с использованием HTML, CSS, JavaScript и автоматического деплоя через GitHub Actions на GitHub Pages.

Возможности

HTML Валидация

Автоматическая валидация HTML для обеспечения корректности разметки.

PostCSS

Современная обработка CSS с автоматической минификацией и оптимизацией.

GitHub Pages

Автоматический деплой на GitHub Pages через GitHub Actions.
